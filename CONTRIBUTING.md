# Contributing to Git Zero-to-Hero

Thanks for wanting to contribute! Here's how:

## How to Contribute

1. **Fork** this repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/Git-Zero-to-Hero.git`
3. **Create a branch**: `git checkout -b feature/your-improvement`
4. **Make changes** and commit: `git commit -m "Add: description of change"`
5. **Push**: `git push origin feature/your-improvement`
6. **Open a Pull Request**

## What You Can Contribute

- Fix typos or improve explanations
- Add more interview questions with answers
- Add real-world examples or use cases
- Improve or add new animated SVG diagrams
- Add translations
- Report issues or suggest improvements

## Commit Convention

Use prefixes:
- `Add:` for new content
- `Fix:` for corrections
- `Update:` for improvements to existing content
- `Docs:` for documentation meta-changes

## Guidelines

- Keep explanations beginner-friendly where possible
- Include command examples with expected output
- Test any commands you add
- Follow existing formatting style

## Questions?

Open an issue — happy to help!
